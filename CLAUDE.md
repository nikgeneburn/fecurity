# Fecurity Lua Port

This project ports the Fecurity web menu into a Roblox Luau library bundled by DarkLua.

Rules:

- Keep files focused and split before they get large.
- Put reusable behavior in `src/` modules, not in examples.
- Keep public demos on the widget API only.
- Keep hosted asset URLs centralized in `src/assets/AssetRegistry.luau`.
- Build with `build.bat` before checking `dist/`.
- Do not replace web assets with unrelated Roblox defaults.
